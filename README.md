# A typing test game for people terminally in the terminal
![keySmash](https://github.com/user-attachments/assets/144e92f9-8757-48ec-8aec-c6757f063653)

## usage
Install ncurses:
```sh
sudo apt-get install libncurses-dev
```
Run make once
```sh
make
```
The binary now exists in build/bin/main.
You can also run the code with
```sh
make run
```
